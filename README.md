# The Ergo Kiwi Simple Archetype

This simple archetype builds a sample maven based project with support for:
 1. Junit 4
 2. Mockito
 3. Hamcrest matchers
 4. Google findbugs
 5. Checkstyle
 6. JaCoCo test coverage
 7. Editor config
 8. Travis 
 9. Codecov.io
